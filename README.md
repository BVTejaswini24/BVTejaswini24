## 💼 Featured Projects

### 🌐 ResQNet — Real-Time Emergency Resource Logistics Network
> A resilient, highly-scalable platform coordinating emergency logistics and supply distribution during natural disasters.

*   **Business Problem Solved:** Disasters suffer from critical logistics bottlenecks due to communication lag. ResQNet reduces emergency response coordination overhead by providing real-time data propagation across supply nodes.
*   **Architecture & Tech Stack:** React, Node.js, Express, MongoDB, WebSocket connection layer, Tailwind CSS.
*   **Key Features:**
    *   Dynamic resource tracking with custom geospatial querying.
    *   Real-time event broadcasts powered by persistent WebSockets.
    *   Optimized database schemas minimizing write contention during high-stress traffic surges.
*   **Engineering Challenges:** Solved a critical WebSocket connection drops issue under concurrent loads by building an exponential backoff reconnection queue, reducing message loss rates to effectively zero.

### 🚗 ParkOptima — Urban Parking Spatial Optimization Analytics
> An analytical infrastructure simulator designed to optimize metropolitan parking allocation.

*   **Business Problem Solved:** Reduces idle fuel waste and traffic congestion by mathematically distributing vehicles to optimal open bays.
*   **Tech Stack:** Python, Pandas, NumPy, Streamlit.
*   **Key Features:**
    *   Predictive parking density modeling based on historical urban flow datasets.
    *   Dynamic routing simulations based on proximity and duration matrices.
